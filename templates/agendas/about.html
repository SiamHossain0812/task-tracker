About the Application

Dr. Niaz's Agenda Tracker is a lightweight task management application designed to help professionals organize their work efficiently. Built with Django and modern web technologies, it provides a clean, intuitive interface for managing tasks, projects, and deadlines.

Key Features:

Global Search
Responsive Design
